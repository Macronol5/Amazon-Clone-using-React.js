Amazon Clone using React JS


        What is React.js?

        It is an open-source JavaScript library that is used for building user interfaces 
        specifically for single-page applications.
 
        What is Node.js?

        Node.js is an open-source, cross-platform, back-end JavaScript runtime environment 
        that runs on the V8 engine and executes JavaScript code outside a web browser.

        What is npm?

        npm is the package manager for the Node JavaScript platform.

        What is npx?

        npx is a very powerful command that's been available in npm,
        npx lets you run code built with Node. js and published through the npm registry.
